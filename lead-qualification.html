// overview

This is not a basic chatbot. It is a live AI agent deployed on this portfolio that handles the full visitor-to-lead pipeline autonomously. The agent converses naturally with visitors, understands context across multiple pages, detects hiring intent without trigger words or buttons, and qualifies leads through conversation — not forms.

When a visitor shows interest, the agent naturally collects their name, project, and email through conversation. The moment both are confirmed, a 10-step automated pipeline fires — logging the lead to Google Sheets, sending an instant notification, and delivering a personalized follow-up email to the lead — all before the conversation even ends.

Built entirely with free tools and no paid courses — Groq AI for the language model, Cloudflare Workers as the secure backend proxy, and Google Apps Script for the automation pipeline.

// full vision

This agent architecture can be adapted for any business that receives inquiries — service providers, freelancers, small businesses, agencies. The same pattern applies: visitor arrives → agent qualifies → lead captured → pipeline fires → owner notified → follow-up sent.

The same stack — Groq + Cloudflare Workers + Google Apps Script — can be extended to include CRM integration, SMS notifications, calendar booking, or multi-agent orchestration for more complex qualification flows.
