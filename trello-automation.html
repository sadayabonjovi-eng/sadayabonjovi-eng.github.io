// overview

This is not a simple Trello board. It is a fully automated client onboarding pipeline where every stage transition is handled by Trello's Butler automation — no manual checklist swapping, no forgetting to update due dates, no missed follow-ups.

When a new client card is created, the automation fires immediately — setting a due date, description, checklist, and email notification in one rule. As the client progresses through each stage, the checklist swaps automatically, the label updates, a comment is logged with the date, and the due date resets for the next stage.

Built entirely with Trello's free Butler automation — no external tools, no paid plans, no code required.

// automation rules

Rule 01 — New Lead
When a card is added to list "Lead" → set due date 7 days · set description · send email notification to sadayabonjovi@gmail.com · add "Lead Steps" checklist from template
Rule 02 — Urgent Flag
When due date is less than 2 days from now → move card to top of "URGENT!!!!!" list
Rule 03 — Urgent Label
When a card is added to list "URGENT!!!!!" → add dark red label to the card
Rule 04 — Stage Transition (checklist-driven)
When checklist [X Steps] is completed → move card to top of list [Next Stage] · covers Lead → Proposal Sent → Contract Signed → Onboarding → Active → Offboarding
Rule 05 — Label on Stage Entry
When a card is moved into list "Active" → add green label · pattern repeated per stage with stage-specific label colors
Rule 06 — Audit Comment
When a card is moved into a list → post comment "[stage name] — [date]" for audit trail and visibility
Rule 07 — Due Date Reminder
When due date is 1 day from now → send email to owner with subject "ALERT!!!!" and message "DUE DATE IS NEAR"
Rule 08 — Pipeline Complete
When checklist "Offboarding Steps" is completed → move card to top of "Completed" list
